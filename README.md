# üìñ Cara Penggunaan
Authentication: Tunggu animasi booting selesai, baca panel dukungan, lalu masukkan Nama Anda sebagai Operator sistem.

- Targeting: Pilih Menu 1 (Initialize Target). Paste atau ketik domain target (Contoh: google.com). Tunggu hingga muncul indikator TARGET LOCK.
- Scanning: Masuk ke Menu 2 (Command Center). Pilih ID modul dari 1 hingga 15 untuk memulai investigasi otomatis.
- Reporting: Setelah selesai, kembali ke menu utama dan pilih Menu 3 (Report Center) untuk mengunduh laporan pemindaian profesional Anda.
